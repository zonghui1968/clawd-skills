### 🐛 常见问题
**Q: 游戏无法启动？**
A: 确保使用现代浏览器（Chrome 90+, Firefox 88+, Safari 14+）

**Q: 方块显示不正常？**
A: 刷新页面，清除浏览器缓存

**Q: 移动端按钮不响应？**
A: 关闭浏览器的缩放功能，使用横屏模式

## 游戏二：你画我猜（Draw & Guess）

### 📍 游戏位置
```
C:\Users\宗晖\clawd\draw-and-guess\
```

### 🚀 运行方法

#### 步骤1：启动游戏服务器
游戏已经自动启动在端口 **3001**！

**验证服务器运行状态：**
打开浏览器访问：http://localhost:3001

如果看到 "你画我猜" 游戏界面，说明服务器正常运行！

#### 步骤2：开始游戏
1. 打开浏览器访问：**http://localhost:3001**
2. 输入你的名字
3. 点击"创建房间"或"加入房间"
4. 分享房间号给朋友（双人游戏需要两个浏览器窗口）

#### 测试方法（单人测试）
1. 打开两个浏览器窗口
2. 两个窗口都访问 http://localhost:3001
3. 窗口1：输入玩家A的名字，创建房间
4. 窗口2：输入玩家B的名字，加入房间（输入窗口1显示的房间号）
5. 开始游戏！

### 🎯 游戏玩法
1. **创建房间**：生成6位房间号
2. **加入房间**：输入房间号加入
3. **游戏流程**：
   - 系统随机选择一个单词
   - 玩家A看到单词并开始绘画
   - 玩家B根据绘画猜单词
   - 限时60秒
   - 猜对得分，角色交换

### ✅ 功能特性
- 实时多人游戏（2名玩家）
- WebSocket 通信
- 绘画画板工具（画笔、颜色、橡皮擦、清空）
- 实时同步绘画
- 计时系统（60秒）
- 计分系统
- 房间系统（随机6位房间号）
- 单词库（60+ 个单词）

### 🛠️ 服务器管理

#### 查看服务器状态
服务器应该已经在后台运行。检查方法：
```bash
netstat -ano | findstr :3001
```

#### 重启服务器
如果需要重启服务器：
```bash
cd C:\Users\宗晖\clawd\draw-and-guess
npm start
```

#### 停止服务器
找到进程并终止：
```bash
# 查找进程ID
netstat -ano | findstr :3001

# 终止进程（替换 <PID> 为实际的进程ID）
taskkill /PID <PID> /F
```

### 🐛 常见问题

**Q: 无法连接到服务器？**
A: 确保服务器正在运行，访问 http://localhost:3001

**Q: 端口 3001 被占用？**
A: 修改 `src/server.js` 中的端口号，然后重启服务器

**Q: 绘画不显示？**
A: 刷新页面，确保浏览器支持 Canvas

**Q: 无法加入房间？**
A: 确保房间号输入正确，检查浏览器控制台是否有错误
